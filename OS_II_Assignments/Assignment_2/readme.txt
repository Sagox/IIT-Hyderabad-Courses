2)To compile the file Assgn1-RMSCS17BTECH11034.cpp use:

g++ Assgn1-RMSCS17BTECH11034.cpp -o Assgn1-RMSCS17BTECH11034
______________________________________________________________

Executing the program

Before executing there must be a file called inp-params.txt
present in the same directory as the files.

1)To execute the program Assgn1-RMSCS17BTECH11034.cpp use:

./Assgn1-RMSCS17BTECH11034

2)To execute the program Assgn1-EDFCS17BTECH11034.cpp use:

./Assgn1-EDFCS17BTECH11034

_______________________________________________________________

Output

The output of either program is two files,
1)EDF/RMS-Log.txt
2)EDF/RM-Stats.txt

File 1 contains logs of all events during scheduling.
File 2 has statistics related to the scheduling.%
